# Ethiopian Chronicles 🏛️

An interactive, AI-narrated journey through Ethiopian history — from the Kingdom of Aksum to the Battle of Adwa and beyond. Built with Flutter.

## Features
- **Interactive Timeline** — Scroll through 2,000 years of Ethiopian history
- **AI Narration** — Every chapter narrated with natural voice synthesis
- **Battle Maps** — Animated troop movements for key battles
- **Dual Language** — English and Amharic
- **Gamification** — Unlock achievements as you explore
- **Offline-first** — Download content, no data connection needed

## Architecture
- **Framework:** Flutter 3.x (Impeller rendering)
- **Content:** JSON-driven chapter system
- **Audio:** edge-tts / ElevenLabs narration
- **State:** Riverpod
